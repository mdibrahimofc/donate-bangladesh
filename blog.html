What is the Document Object Model (DOM)?

The Document Object Model DOM is a programming layer for web documents. It represent the structure of a webpage likess a trees of objects allowing developer to intaract with and menipulate HTML and XML documents. Each element (like heading, paragraphs, images) on a webpage becomes a node in the DOM tree, which can be accessed and modified using JavaScript.

How do you select an element from the DOM?

To salect an element from the DOM, i can use diferent methods. getElementById() selects an element by it id, while getElementsByClassName() and getElementsByTagName() target element by clas and tag, respectevely. querySelector() selects the first element metching a CSS selector, and querySelectorAll() selects all matching elements. These method allow interact interaction with the webpage.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a tachnique in the DOM where you attech a single event listener to a parent element to menage event clicked by its child elements. Instead of adding individual event listeners to each child, the event bubbles up to the parent, which can then determine the target of the event using propertie like event.target. This is useful because it improves performance and simplifies code, especially when dealing with dynamically added elements or a largeest number of child elements. It reduces memory usage and ensures that newly created elements are automatically covered by the single listener.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an element's attributes and styles in the DOM using JavaScript. for modify attributes, add setAttribute() to change or add an attribute, getAttribute() to retrieve its value, and removeAttribute() to delete it. For styles, access the style property to directly modify inline CSS, like changing the background color or font size. i can also use classList methods such as add(), remove(), and toggle() to manage CSS classes dynamically. These techniques allow you to update an element's attributes and appearance efficiently.
